# Portfólio Pessoal — Igor Dário Barbosa Da Silva

Site pessoal desenvolvido como atividade prática da disciplina de Desenvolvimento de Software (DS) — CIn/UFPE.

🔗 **Acesse:** https://www.cin.ufpe.br/~idbs/

## 📄 Páginas

- **Início** — apresentação pessoal, curso, interesses e contatos
- **Sobre** — trajetória acadêmica, habilidades e objetivos profissionais
- **Projetos** — projetos desenvolvidos nas disciplinas do curso

## 🛠️ Tecnologias

- React
- Vite
- React Router DOM
- Tailwind CSS

## 🚀 Como rodar localmente

```bash
git clone https://github.com/igordario-hub/portfolio-idbs
cd portfolio-idbs
npm install
npm run dev
```

Acesse em: http://localhost:5173

## 📦 Como gerar o build

```bash
npm run build
```

Os arquivos estáticos serão gerados na pasta `dist/`.
